Module imxrt_dma::channel[−][src]

DMA channels

channel contains the DMA Channel type, along with helper functions for defining transfers.

Channel methods that specify memory involved in transfers are marked unsafe. You must be very careful when calling these methods, particuarly when a channel is already enabled.
